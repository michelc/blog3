Configuration base de données

2005-07-18 #qc

Jusqu'à présent, la sélection de l'identifiant de la base de donnée à utiliser se faisait en dur dans le Global.asax.cs. C'est désormais paramétrable via le web.config de la façon suivante :

  • <add key="URL_http://localhost/qc" value="QcLocal" />
  • <add key="URL_http://localhost:8088/" value="QcXSP" />
  • <add key="URL_http://qctest.monoforge.com/" value="QcTest" />

Ensuite, il suffit comme auparavant de configurer la chaine de connection pour chaque identifiant :

  • <add key="CnxString_QcLocal" value="Provider=Microsoft.Jet.OLEDB.4.0;Data Source=...
  • <add key="CnxString_QcXSP" value="{mysql};Database=quickcontent;Data Source=...
  • <add key="CnxString_QcTest" value="{mysql};Database=qctest;Data Source=...
