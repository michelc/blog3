Création de la brique Integration.AspxPart

2004-10-05 #qc

Permet d'afficher le résultat de l'interprétation/compilation d'un extrait de code ASP.NET. Inspiré du ComponentModule de Rainbow, mais au lieu de stocker le code ASP.NET dans la base, il est chargé depuis un fichier externe.

Utilise les mêmes paramètres que HtmlPart et TextPart.
